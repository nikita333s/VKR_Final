## 📊 Описание работы

Целью данной выпускной квалификационной работы является разработка BI-дашборда для поддержки принятия управленческих решений на уровне отдела продаж.

В рамках работы предполагается создание аналитического инструмента, позволяющего руководителю отдела оценивать эффективность сотрудников, анализировать показатели продаж и выявлять проблемные зоны в работе команды.

---

## 🎯 Предметная область

Отдел продаж компании.  
Основной акцент — анализ эффективности сотрудников (менеджеров по продажам).

---

## 📈 Планируемые метрики

- выручка (общая и в динамике)
- количество сделок
- win rate (доля успешных сделок)
- средний чек
- выручка и сделки по сотрудникам
- скорость закрытия сделок
- анализ воронки продаж
- сравнение эффективности сотрудников

---

## 🧰 Используемые инструменты

- Yandex DataLens — визуализация и дашборды  
- Python (опционально) — обработка данных  
- CSV / Excel — исходные данные  

---

## 📁 Данные

Планируется использование открытого датасета:

- https://www.kaggle.com/datasets/innocentmfa/crm-sales-opportunities  

При необходимости данные могут быть дополнительно обработаны или расширены.

## 🗂 План работы

### Этап 1. Подготовка проекта
- создание репозитория
- формирование плана работы

### Этап 2. Обзор источников
- анализ литературы по BI
- изучение подходов к построению дашбордов
- анализ метрик и KPI

### Этап 3. Работа с данными
- загрузка датасета
- очистка и подготовка данных
- проведение EDA (разведочный анализ данных)

### Этап 4. Разработка дашборда
- определение структуры дашборда
- разработка визуализаций в Yandex DataLens
- реализация интерактивных элементов

### Этап 5. Анализ результатов
- интерпретация показателей
- формирование управленческих выводов

### Этап 6. Оформление ВКР
- написание текста работы
- подготовка к защите

---

## 📌 Ожидаемый результат

Интерактивный BI-дашборд, позволяющий:
- анализировать эффективность сотрудников отдела продаж
- выявлять узкие места в воронке продаж
- принимать управленческие решения на основе данных
